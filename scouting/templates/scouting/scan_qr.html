Camera QR Scanning

For camera-based QR scanning, you would need to implement HTML5 getUserMedia API with a JavaScript QR code library. Since we're using only Python standard library, this feature requires additional client-side JavaScript libraries.

Alternative: Take a photo of the QR code and use an external QR code reader app to extract the text, then paste it in the first tab.

How to Use
  1. Have the scouter open their QR code screen (from offline submission)
  2. Click "Show Raw Data" on their screen to reveal the text data
  3. Copy the entire text data from the textarea
  4. Paste it into the "QR Code Data" field above
  5. Click "Process QR Code" to save the submission
⚠️ Important: Make sure the data is complete and not truncated. The submission will fail if the data is incomplete or corrupted.
